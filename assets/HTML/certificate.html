Programming Introduction (Oracle)

O Curso Programming Introduction (Oracle)... tem o objetivo de capacitar os alunos para as trilhas de certificação do Oracle Academy, iniciando com a formação mais básica, para compreender as perspectivas da lógica e da programação e dando suporte para as trilhas de formação mais específicas e profissionais, por exemplo, Java fundamentals, Java Foundations, Java Programming e etc. Esse projeto terá o acompanhamento sistemático dos alunos Embaixadores Oracle e de professores em jornada da Anima

Carga horária: 40h

Java Foundations - Oracle Academy

O curso Java Foundations... da Oracle Academy é um programa abrangente que introduz os alunos aos conceitos de programação orientada a objetos, terminologia e sintaxe, além dos passos necessários para criar programas Java básicos. O curso cobre tópicos como o processo de desenvolvimento de software, tipos de dados em Java, métodos e classes de biblioteca em Java, declarações de decisão, construções de loop, criação de classes, interação entre objetos e encapsulamento, arrays e exceções, conceitos e técnicas de depuração e uma introdução ao JavaFX. Com uma duração total recomendada de 90 horas, o curso é adequado para estudantes que desejam aprender programação em Java e construir sua experiência em programação orientada a objetos usando Java.

Carga horária: 90h

Assistente de Produção

O curso de Assistente de Planeja...mento, Programação e Controle de Produção oferecido pelo SENAI prepara profissionais para auxiliar na administração e produção de processos industriais. O curso segue normas e procedimentos técnicos de qualidade, segurança, higiene e saúde. É uma excelente opção para quem deseja aprimorar suas habilidades no campo da produção industrial.

Carga horária: 476h

Diploma Técnico em Comércio

O curso de Técnico em Comércio é... um programa educacional integrado ao ensino médio, oferecido pelo Centro Estadual de Educação Profissional em Gestão e Meio Ambiente. Este curso está sob o Eixo Tecnológico de Gestão e Negócios e foi concluído no ano de 2019. A formação técnica em Comércio capacita o aluno com habilidades e conhecimentos necessários para navegar no mundo dos negócios e do comércio. Este diploma confere ao aluno os direitos e prerrogativas para uma carreira no setor de comércio.

Carga horária: 4.440h
