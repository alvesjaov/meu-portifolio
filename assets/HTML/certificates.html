Tech in Microsoft

Esse é um projeto de extensão que... tem o intuito de inserir os alunos nas trilhas de formação da Microsoft tanto com o objetivo de aprofundar os conhecimentos nas tecnologias Microsoft, requeridas pelo mercado de trabalho, quanto de capacitar nossos alunos para inovarem e buscarem soluções diferenciadas na área de TI&C com tecnologias dessa big tech. Além disso, esse projeto será um efetivo apoio para a compreensão de competências relativas à inteligência artificial, que são requeridas na UC de Inteligência Artificial de TI&C.

Carga horária: 60h

AWS academy

Currículo de computação na nuvem... com cursos nas categorias: fundamentos de nuvem, operações de nuvem, arquiteturas e desenvolvimento de nuvens e, análise de dados em nuvem.

Carga horária: 60h

Java Foundations - Oracle Academy

O curso Java Foundations... da Oracle Academy é um programa abrangente que introduz os alunos aos conceitos de programação orientada a objetos, terminologia e sintaxe, além dos passos necessários para criar programas Java básicos. O curso cobre tópicos como o processo de desenvolvimento de software, tipos de dados em Java, métodos e classes de biblioteca em Java, declarações de decisão, construções de loop, criação de classes, interação entre objetos e encapsulamento, arrays e exceções, conceitos e técnicas de depuração e uma introdução ao JavaFX. Com uma duração total recomendada de 90 horas, o curso é adequado para estudantes que desejam aprender programação em Java e construir sua experiência em programação orientada a objetos usando Java.

Carga horária: 90h
